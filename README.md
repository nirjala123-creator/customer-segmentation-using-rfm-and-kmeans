# customer-segmentation-using-rfm-and-kmeans
Customer behavior analysis using RFM framework and KMeans clustering to identify high-value and at-risk customers.
Customer Segmentation using RFM & KMeans

Project Overview

This project analyzes customer purchasing behavior using the RFM (Recency, Frequency, Monetary) framework. Customers are segmented into meaningful groups using KMeans clustering to identify high-value, regular, and at-risk customers.

 Objectives

Perform exploratory data analysis (EDA)

Apply RFM analysis to evaluate customer behavior

Use KMeans clustering for customer segmentation

Generate business insights and recommendations
